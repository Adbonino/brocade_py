cfgadd - Agregar zonas a una configuración

cfgadd: agrega uno o más miembros a una configuración de zona existente. Este comando modifica la configuración definida.

Para que el cambio surta efecto, habilite la configuración con el comando cfgEnable. Para que el cambio se conserve tras reiniciar el switch, guarde la configuración en la memoria no volátil con el comando fgSave. La lista de miembros del alias debe tener al menos un miembro (no se permiten listas vacías y no puede ser otro alias).

La lista de miembros del alias debe tener al menos un miembro (no se permiten listas vacías).

Sintaxis: cfgadd "cfgName", "member[; member...]"

Se requieren los siguientes parámetros:

Parámetro Descripción
"cfgName" Especifique un nombre para la configuración de la zona, entre comillas dobles.
"member" Especifique un miembro de zona o una lista de miembros de zona que se añadirán a la configuración. La lista debe estar entre comillas dobles. Los miembros deben estar separados por punto y coma.

Ejemplos de uso:

   
    # Para agregar dos nuevas zonas a la configuración "Test_cfg": 
    switch:admin> cfgadd "Test_cfg", "greenzone;bluezone"
