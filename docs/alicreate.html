alicreate - Creación de alias

alicreate: crea un nuevo alias de zona. La lista de miembros del alias debe tener al menos un miembro (no se permiten listas vacías y no puede ser otro alias).

Este comando cambia la configuración definida. Para que el cambio surta efecto, habilite la configuración de zona con el comando cfgEnable. Para que el cambio se conserve tras reiniciar el switch, guarde la configuración de zona en la memoria no volátil con el comando cfgSave.

Sintaxis: alicreate "aliName","member[; member...]"

Se requieren los siguientes parámetros:

Parámetro Descripción
"aliName" Especifica un nombre para el alias, entre comillas dobles. El nombre debe comenzar con una letra y puede incluir letras, números y guiones bajos. Es sensible a mayúsculas/minúsculas. Ejemplos: "Ali_1" y "ali_1" son diferentes. Los espacios se ignoran.
"member" Especifica uno o más miembros que se agregarán al alias, entre comillas dobles y separados por punto y coma. Un miembro puede ser:
  • Un par de dominio de switch e índice de puerto (switchShow muestra los índices válidos).
  • Un WWN (World Wide Name).

Ejemplos de uso:

   
    # Para crear un alias de zona definido por pares de índice de dominio y puerto: 
    switch:admin> alicreate "array1", "2,32; 2,33; 2,34"

    # Para crear un alias de zona con un miembro definido por WWN.
    switch:admin> alicreate "array2", "21:00:00:20:37:0c:66:23"
