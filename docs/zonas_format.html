Zonas - converción de formatos

zonas_format.py:

Script de Python que recibe como entrada un archivo .csv con los datos de las zonas y sus correspondientes miembros y permite obtener como salida un archivo en formato .yaml.

Ejemplo de uso

             
            > python .\zonas_format.py -h
              usage: zonas_format.py [-h] csv_in yaml_out  

              Convertir CSV de zonas a YAML con indentación

              positional arguments:
                csv_in      Archivo CSV de entrada
                yaml_out    Archivo YAML de salida

              options:
                -h, --help  show this help message and exit
            
            > #ejemplo
            > python zonas_format.py zonas.csv zonas.yaml
            Archivo YAML generado: alias.yaml

Parámetros

Nombre Tipo Descripción
csv_in String Path relativo o absoluto del archivo .csv de entrada con los datos de la zonas y miembros (Ver formato .csv)
yaml_out String Path relativo o absoluto del archivo .yaml de salida en donde se guardaran los datos de las zonas y sus miembros en formato .yaml

Fomato de Archivo .cvs de entrada

        
      Nombre_zona,miembro[, miembro...]

      zona1,miembro1,miembro2,....,miembroX
      zona2,miembro1,miembro2,....,miembroX
      ...

      # ejemplo
      # cat zonas.csv
      AIX72_ORCLX_DESA_v1_fcs4_V7200,AIX72_ORCLX_DESA_v1_fcs4,V7200CONTI_N1_P2,V7200CONTI_N2_P1
      AIX72_ORCLX_DESA_v1_fcs4_V9000,AIX72_ORCLX_DESA_v1_fcs4,SVC_V9000_N1_PORT_7_2,SVC_V9000_N2_PORT_7_2
      AIX72_ORCLX_DESA_v2_fcs4_V7200,AIX72_ORCLX_DESA_v2_fcs4,V7200CONTI_N1_P2,V7200CONTI_N2_P1
      ...
Nombre Descripción
Nombre_zona Especifica el nombre que se le asigna a la zona
miembro Alias o WWN del miembro que formará parte de la zona, puede ser uno o varios separados por coma
separador todos los dartos usan como separador: ","
